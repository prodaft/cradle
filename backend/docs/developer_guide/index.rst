Developer Guide
===============

The developer guide provides a detailed description of the code structure and
the architecture of the tool. The developer guide provides a detailed description 
of the code structure and the architecture of the CRADLE backend. It presents the 
terminology of the application. Then, it provides an in-depth explanation of the 
access control. Furthermore, it presents the application structure of the project and 
the logging functionality. Finally, it provides a guideline to writing tests inside the
application.
